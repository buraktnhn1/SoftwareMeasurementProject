# ISO 15939 Measurement & Analysis Simulator

This project is a Java Swing-based application designed to simulate the **ISO 15939** software measurement and analysis process. It provides a guided, step-by-step interface for users to define quality scenarios, plan measurements, collect raw data, and generate analytical reports.

## 🚀 Key Features
*   **Step-by-Step UI Flow:** A 5-stage guided process built using `CardLayout` architecture for seamless transitions.
*   **Dynamic Scenario Management:** Metrics and data entry fields adapt based on the selected Quality Type (Product/Process) and Mode (Health/Education).
*   **Real-time Analysis:** Implements normalization and weighting algorithms to transform raw input values into actionable quality scores.
*   **Visual Progress Indicator:** A custom navigation bar to track current progress through the measurement lifecycle.

## 🛠 Tech Stack
*   **Language:** Java 21+
*   **GUI Library:** Java Swing & AWT
*   **Version Control:** Git
*   **Architecture:** Modular View-based Design

## 📂 Project Structure
*   `src/view`: Contains all UI components and panels (`ProfilePanel`, `DefinePanel`, `PlanPanel`, `CollectPanel`, `AnalysePanel`).
*   `src/controller`: Reserved for handling the business logic and inter-panel communication.
*   `src/model`: Defined for future scalability of data structures and metric definitions.

## 📖 How to Run
1.  Clone the repository:

    ```
